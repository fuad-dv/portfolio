## 👤 About

This is the personal portfolio of **MD Fuad Hassan Shiraji**, a Software Engineering undergraduate at **Daffodil International University (DIU)**, passionate about **Machine Learning, AI**, and building intelligent systems for real-world problems.

---

## ✨ Features

- 🎬 **Terminal-style loading animation** on page open
- 📸 **Embedded profile photo** (no external image hosting needed)
- 🌙 **Dark techy theme** with neon cyan & purple accents
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- ⚡ **Single file deployment** — just one `index.html`, no build tools
- 🔗 **Clickable certificates** — IELTS & Adobe Premiere Pro
- 📄 **Resume button** — direct link to view/download CV
- 🎞️ **Smooth scroll reveal animations** throughout

## 🚀 Deployment

This site is deployed via **GitHub Pages**.

To run locally, simply open `index.html` in any browser — no server needed.

```bash
# Clone the repo
git clone https://github.com/fuad-dv/portfolio.git

# Open in browser
open index.html
```
